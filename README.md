# Android_Cai_Client
## Android_Cai客户端代码总结
<p>前端页面主要使用了RelativeLayout、LinearLayout布局</p>
<p>ListView、GridView、TabHost视图控件</p>
<p>Fragment页面转换</p>
<p>Intent页面跳转</p>
<p>SQLite进行微量页面数据传递(例如保存注册信息等)</p>
<p>SmartRefreshLayout解决缓冲池过慢对应用的影响实现上下拉加载</p>
<p>Glide使得应用图片对应用影响进度降低</p>
<p>游标的使用</p>
<p>JPush的集成,实现脱离服务器端发送APP信息</p>
